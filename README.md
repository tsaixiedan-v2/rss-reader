# rss-reader

A Feedly-like RSS feed reader web app built with TypeScript, React, and Node.js.

## Tech Stack

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express
- **Database**: SQLite
- **RSS Parsing**: rss-parser

## Getting Started

```bash
npm install
npm run dev
```
